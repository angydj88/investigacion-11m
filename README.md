# Terrorismo Investigación 11-M

Proyecto de documentación e investigación sobre los atentados del 11 de marzo de 2004 en Madrid. Base de conocimiento estructurada con timeline cronológico, análisis documental, fuentes verificadas y datos estructurados.

## Datos clave

- **Fecha:** 11 de marzo de 2004
- **Lugar:** Cercanías de Madrid (4 trenes)
- **Víctimas mortales:** 192
- **Heridos:** ~2.000
- **Sentencia:** Audiencia Nacional, 31 de octubre de 2007

## Metodología

Este proyecto recopila información exclusivamente de **fuentes públicas y oficiales**:
- Sentencias judiciales publicadas
- Informes de comisiones parlamentarias
- Publicaciones periodísticas de medios de referencia
- Libros y publicaciones académicas
